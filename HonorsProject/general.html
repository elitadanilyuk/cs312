General Resources

Live Front-End Development

There are various resources front-end developers can use to view their front end development changes live.

CodeSandbox

Cost: Free & Subscription Options
Popular Supported Languages: JavaScript, TypeScript, React, NodeJS, Angular, HTML, CSS

CodeSandbox is an online development environment that allows prototyping, creating, and experimenting with web applications.

CodeSandbox has built-in Hot Module Replacement/Reloading (HMR) which will refresh your web application with any live changes. This allows for faster prototyping and experimenting to test any new code changes. Additionally, CodeSandbox is integrated with GitHub which allows you to import existing repositories and syncs any changes automatically. This integration also allows the ability to create commits, open PRs, and export sandboxes.

Free version of CodeSandbox:

  • unlimited number of public sandboxes
  • access to any public npm packages
  • personal dashboard and a public profile
  • live web application reload
  • collaborative code editing
  • classroom mode to allow single control
  • real time in-editor chat

To use CodeSandbox, you will need to create an account to begin creating and developing in code sandboxes. Read more HERE, to learn more about what CodeSandbox offers, including its subscription differences.

CodePen

Cost: Free & Subscription Options
Popular Supported Languages: HTML, CSS, JavaScript

CodePen is another online development environment that allows testing and showing user-created HTML, CSS, and JavaScript code.

CodePen is a great option for front-end developers and designers who are new to development and plan on creating 'simple' websites. It is a development environment as well as an oline learning community for testing and showcasing work; developers create, test, and deploy code snippets known as pens. CodePen, similarly with CodeSandbox, refreshes your web site with live changes as you save your work.

Free version of CodePen:

  • unlimited number of public Pens and Collections
  • unlimited templates
  • personal dashboard and public profile
  • live browser reload

To use CodePen, you will need to create an account to begin creating and developing your Pens and collections. Read more HERE, to learn more about what CodePen offers. For subscription differences, go HERE.

VScode Live Server Extension

Cost: Free
Popular Supported Languages: JavaScript, TypeScript, React, NodeJS, Angular, HTML, CSS, PHP

Live Server is a Visual Studio Code (VScode) extension, thus, a great option for developers using VScode.

The VScode Live Server Extension is a great overall option for live front-end development and design. As you save your work in VScode the Live Server Extension reloads your live browser with any new changes.

VScode Extension:

  • compatible with VScode languages
  • hot key control
  • Quick Development Live Server with a live browser reload
  • customizable Port Number, Server Root, browser
  • support for any file, including dynamic pages

To use the VScode Live Server Extension you will need to have VScode installed, then search for Live Server and install the extension. Read more HERE, to learn more about VScode. To read more about the extension go HERE
